About This Project

This frontend now runs on Flask and connects to a Python hybrid recommendation engine.

Content-Based

TF-IDF converts cleaned product text into vectors. Cosine similarity then finds products that look alike.

Semantic AI

Sentence embeddings try to understand meaning-level similarity, so products with similar intent can match even when exact keywords differ.

Profile + Popularity

The final system builds a preference profile from manually liked seed items and blends that with a popularity prior because this dataset does not contain real user histories.
